## *<a name="html">HTML基本标签</a>*

>####什么是HTML 标签
>* HTML 文档和 HTML 元素是通过 HTML 标签进行标记的
>* HTML 标签由开始标签和结束标签组成
>* 开始标签是被括号包围的元素名
>* 结束标签是被括号包围的斜杠和元素名
>* 某些 HTML 元素没有结束标签，比如 <br />

>####HTML 标题
>HTML 标题（Heading）是通过 ``` <h1> - <h6>```  等标签进行定义的。

>####HTML 段落
>HTML 段落是通过 <p> 标签进行定义的。
